# 💰 Personal Finance Dashboard - Zorvyn Assignment

A robust Spring Boot application for tracking personal finances with built-in **Role-Based Access Control (RBAC)**.

## 🚀 Features

- **RBAC Integration:** - **ADMIN:** Full permissions (Add, View, and Delete transactions).
  - **ANALYST:** View-only access with dashboard insights.
  - **VIEWER:** Basic read-only access.
- **Real-time Calculations:** Automatically calculates Net Balance based on Income/Expense entries.
- **Dynamic UI:** Responsive dashboard that hides/shows features based on the selected User Role.
- **Clean Architecture:** Follows the Controller-Service-Repository pattern.

## 🛠️ Tech Stack

- **Backend:** Java 17+, Spring Boot, Spring Data JPA
- **Database:** MySQL / PostgreSQL
- **Frontend:** HTML5, CSS3 (Bootstrap 5), JavaScript (Fetch API)
